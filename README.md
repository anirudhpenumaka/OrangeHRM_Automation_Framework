# OrangeHRM Automation Framework

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)

## Features Implemented
- Config-driven browser and URL management
- WebDriver lifecycle handled via BaseTest
- Login module automation
  - Valid login
  - Invalid login
  - Blank credentials validation
  - UI validations (page title, URL, password masking)

## Framework Highlights
- Clean Page Object Model design
- Independent test execution using TestNG lifecycle
- Scalable structure for adding more modules

## Upcoming Enhancements
- Logging (Log4j2)
- Cross-browser execution
- Dashboard & Logout automation
- Admin user management flows
